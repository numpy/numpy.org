• Numpy array forms the core of the organically growing numeric Python array library ecosystem that now supports GPUs, sparse, distributed arrays and more.

    Several of these newer libraries such as CuPy, Sparse and Dask, implement the NumPy API adding support for modern user cases, newer hardware and higher scalability of array computing. Other array libraries such as Xarray, TensorLy consume NumPy API and build newer functionality on top of it, thus enhancing array computing in Python beyond Numpy capabilities.

    Array Library Capabilities & Application areas
    Xarray Labeled, indexed multidimensional arrays for advanced analytics and visualization
    uarray Python backend-system that decouples array computing library API definitions from implementation to enhance NumPy code capabilities.
    XND Container type that maps Python values to typed memory to accelerate financial, insurance and scientific computing.
    xframe Multi-dimensional labelled array expressions and powerful abstractions for operating, analyzing large datasets.
    Sparse Multi-dimensional arrays that integrate with SciPy-sparse, CuPy and Dask for Machine Learning and data analytics.
    Dask Distributed arrays, parallel computing for data analytics in geo-science, banking, astronomy, satellite imagery and mobile network modeling.
    CuPy NumPy-compatible matrix library accelerated by CUDA used to implement Neural Networks for Deep Learning.
    Tensor.ly Tensor learning, algebra and backend to seamlessly use NumPy, MXNet, PyTorch, TensorFlow or CuPy for ML recommendation systems.
    CuDF GPU DataFrame library built on the Apache Arrow columnar memory format for loading, joining, aggregating, filtering, and data wrangling.
    Arrow Cross-language platform that combines columnar data structure with in-memory data for streaming analytics and interactive visualizations.
    xtensor Multi-dimensional arrays with broadcasting and lazy computing for numerical analysis.
